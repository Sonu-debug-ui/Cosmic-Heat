# 🌌 Cosmic Heat

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue.svg)](https://www.python.org/)
[![Pygame-ce](https://img.shields.io/badge/Pygame--ce-2.5.2%2B-green.svg)](https://pyga.me/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**Cosmic Heat** is an intense, retro-inspired space shooter built with Python and Pygame-ce. Navigate through treacherous star systems, upgrade your arsenal, and defend the galaxy against a relentless mechanical onslaught.

---

## 🚀 The Fleet

Unlock and master different spacecraft, each with unique attributes:

*   **Standard**: The reliable workhorse of the fleet. Well-balanced for any mission.
*   **Neon Blade**: A high-agility interceptor designed for pilots who value speed and precision.
*   **Crimson Ghost**: Built for heavy combat with reinforced plating and superior firepower.
*   **Omega Wing**: The ultimate evolution in ship technology, featuring cutting-edge energy weapons.

## 🔥 Key Features

*   **Dynamic Combat**: Face off against three distinct bosses with unique attack patterns and bullet hell stages.
*   **Environmental Hazards**: It's not just enemies! Dodge massive meteors, survive black hole pulls, and navigate through wormholes.
*   **Arsenal Upgrades**:
    *   **Health & Bullet Refills**: Stay in the fight longer.
    *   **Double Refills**: Boost your scoring potential.
    *   **Fire Rate Upgrades**: Unleash a torrent of destruction.
*   **Leaderboard**: Track your best runs with the integrated high-score system.
*   **Controller Support**: Full joystick/gamepad integration for a true arcade experience.

## 🕹️ Controls

| Action | ⌨️ Keyboard | 🎮 Joystick |
| :--- | :--- | :--- |
| **Movement** | Arrow Keys / WASD | Left Analog Stick |
| **Primary Fire** | `Space` | `Button 0` (A/X) |
| **Pause Game** | `P` | `Options / Menu` |
| **Exit** | `Esc` | - |

## 🛠️ Installation

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/sonu-debug-ui/cosmic-heat.git
    cd cosmic-heat
    ```

2.  **Set Up the Environment**:
    It's recommended to use a virtual environment:
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows: venv\Scripts\activate
    ```

3.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

## 🚀 Quick Start

Launch the game by running the following command in your terminal:
```bash
python main.py
```
